# Student Rating System

Бұл жоба студенттердің рейтинг жүйесін жасауға арналған.

## Мақсаты
Сұрыптау алгоритмдерін қолдану және Java тілінде Comparator және Comparable интерфейстерін пайдалану.

## Сипаттамасы
Бағдарлама студенттердің тізімін GPA немесе балл бойынша сұрыптайды.

## Қолданылған технологиялар
- Java
- ArrayList
- Collections.sort()
- Comparable
- Comparator
